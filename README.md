# xlsx-to-json

To install:

```sh
pip install -r requirements.txt
# or
python3 -m pip install -r requirements.txt
# or
pip install -r requirements.txt --user
# or
sudo pip install -r requirements.txt
# or ...
```

To run:

```sh
python3 main.py
# or
python3 main.py data/urban-girls-movement-labb-1.xlsx
```
